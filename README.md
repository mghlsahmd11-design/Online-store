# NexusTech Landing Page

This is a pure HTML, CSS, and Vanilla JavaScript professional landing page for an Arabic tech company "NexusTech".

## Features
- Pure Vanilla Stack (No React, No Vite, No build tools needed).
- Fully Responsive & RTL support (Arabic).
- Dark & Light Mode with LocalStorage persistence.
- Glassmorphism UI Design.
- Scroll Animations & Counters.

## How to use and deploy (GitHub Pages)
1. Clone or download this repository.
2. Upload the files (`index.html`, `style.css`, `script.js`, and `README.md`) directly to your GitHub repository root.
3. Go to **Settings** > **Pages** in your GitHub repository.
4. Under "Source", select the `main` or `master` branch.
5. Click "Save". Your website will be live in a few minutes!

## Customization
- To change images, place your images in an `images/` directory and update the `src` paths in `index.html`.
- Colors can be adjusted by changing the CSS variables at the top of `style.css`.
